## 🚀 What is Kido?

**Kido** is a modern, lightweight, and intuitive programming language designed specifically for the next generation of creators. It bridges the gap between block-based coding (like Scratch) and complex industrial languages, making "real code" accessible through natural English keywords.

### ✨ Why Kido?

- 🦄 **Natural Syntax:** Use `remember` instead of `var`, `yes/no` instead of `true/false`.
- ⚡ **Full Featured:** Supports functions (`fun`), lists, dictionaries, and file I/O.
- 🛡️ **Safety First:** Built-in **Security Guards** to prevent infinite loops and unauthorized file access.
- 📚 **Curriculum Ready:** Comes with **500+ built-in problems** and examples.

## 🏁 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/animeshdindapersonal-jpg/kido.git
cd kido/kido_sdk

# Run the installer
./install.sh
```

### Your First Program (`hello.kd`)

```kido
# This is a Kido program
print "Hello World!"

remember name = "Aarav"
remember age = 8

print "My name is" name "and I am" age "years old."

if age bigger 5 {
    print "You are a Kido Explorer! 🚀"
}
```

Run it using the CLI:
```bash
kido hello.kd
```

## 🛠️ Project Structure

- **`kido_core/`**: The heart of the language (Lexer, Parser, Interpreter).
- **`kido_cli/`**: Command-line interface for running and debugging.
- **`examples/`**: A massive library of programs including a **500-problem curriculum**.
- **`stress_tests/`**: Complex algorithms (Sieve of Eratosthenes, Fibonacci, etc.) implemented in Kido.

## 💡 Language Comparison

| Feature | Block Coding | **Kido** | C++ / Java |
| :--- | :---: | :---: | :---: |
| **Logic Building** | ✅ | ✅ | ✅ |
| **Variable Syntax** | Drag-n-Drop | `remember x = 10` | `int x = 10;` |
| **Loops** | Blocks | `repeat 5 times` | `for(int i=0;i<5;i++)` |
| **Security** | Sandboxed | **Active Guards** | None |
